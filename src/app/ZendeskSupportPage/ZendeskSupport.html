In App Browser

Launches in app Browser

https://github.com/apache/cordova-plugin-inappbrowser

Installation

ionic cordova plugin add cordova-plugin-inappbrowsernpm install @ionic-native/in-app-browser
Ionic Native Enterprise comes with fully supported and maintained plugins from the Ionic Team. Learn More or if you're interested in an enterprise version of this plugin Contact Us

Supported Platforms

  • AmazonFire OS
  • Android
  • Browser
  • iOS
  • macOS
  • Windows

Usage

import { InAppBrowser } from '@ionic-native/in-app-browser/ngx';
    
    constructor(private iab: InAppBrowser) { }
    
    
    ...
    
    
    const browser = this.iab.create('https://ionicframework.com/');
    
    browser.executeScript(...);
    
    browser.insertCSS(...);
    browser.on('loadstop').subscribe(event => {
       browser.insertCSS({ code: "body{color: red;" });
    });
    
    browser.close();
